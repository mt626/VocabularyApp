# Excel 单词文件格式规范

## 概述

本文档详细描述 VocabularyApp 支持的 Excel 单词导入文件格式，确保用户能够正确准备单词数据文件，实现批量导入功能。

## 文件格式要求

### 文件类型
- 支持格式：`.xlsx`（Excel 2007 及以上版本）
- 文件编码：UTF-8
- 单个文件建议单词数量：100-1000 个

### 列结构

| 列序号 | 列名 | 数据类型 | 必填 | 说明 |
|--------|------|----------|------|------|
| A | 单词列表 | 字符串 | 是 | 英语单词拼写 |
| B | 音标 | 字符串 | 否 | 国际音标 |
| C | 释义 | 字符串 | 是 | 中文释义及词性 |
| D | 词频排序 | 数字 | 否 | 导入时忽略 |

### 数据格式规范

#### 单词列表（A列）
- 纯英文单词拼写，不含空格或特殊字符
- 导入后自动转换为小写格式
- 重复单词会自动去重

#### 音标（B列）
- 支持国际音标符号
- 支持英式音标和美式音标
- 多个音标用分号 `;` 分隔
- 示例：`[ðə;ði:]`

#### 释义（C列）
- 包含词性标注和中文释义
- 词性缩写规范：`n.`, `v.`, `adj.`, `adv.`, `prep.`, `conj.` 等
- 多个释义用分号 `;` 分隔
- 示例：`art. 这；那；adv. 更加`

## 文件模板

### 标准格式示例

```
| 单词列表 | 音标              | 释义                                          | 词频排序 |
|----------|-------------------|-----------------------------------------------|----------|
| the      | [ðə;ði:]          | art. 这；那；adv. 更加（用于比较级，最高级前） | 1        |
| be       | [bi:]             | prep. 在，存在；是                             | 2        |
| and      | [ənd; (ə)n; ænd]  | conj. 和，与；而且；然后；就；但是             | 3        |
| of       | [ɒv; (ə)v]        | prep. 属于；关于；...的；由...组成的           | 4        |
| a        | [ə; eɪ]           | art. 一；任一；每一                            | 5        |
| in       | [ɪn]              | prep. 在...之内；从事于；按照（表示方式）      | 6        |
| to       | [tə;tu;tu:]       | adv. 向前；（门等）关上                       | 7        |
```

### 最小格式示例

仅包含必填字段的简化格式：

```
| 单词列表 | 音标 | 释义                  | 词频排序 |
|----------|------|-----------------------|----------|
| apple    |      | n. 苹果               |          |
| book     |      | n. 书；v. 预订        |          |
| happy    |      | adj. 快乐的           |          |
```

## 导入流程

### 操作步骤

1. **准备文件**: 按照上述格式准备 Excel 文件（`.xlsx`）
2. **传输文件**: 将文件复制到设备存储目录
3. **选择导入**: 在应用中点击"导入单词"按钮
4. **选择文件**: 从文件选择器中选择目标 Excel 文件
5. **确认导入**: 系统自动解析并导入数据
6. **查看结果**: 导入完成后显示成功导入的单词数量

### 导入规则

- **标题行处理**: 第一行作为标题行，不会被导入
- **必填校验**: 单词列和释义列不能为空，否则该行跳过
- **数据清洗**: 自动去除首尾空格和特殊字符
- **重复处理**: 相同单词自动去重，保留最新导入记录
- **错误处理**: 格式错误的行会被跳过，不影响其他数据导入

## 兼容性说明

### 支持的格式
- ✅ Excel 2007-2019 (.xlsx)
- ✅ Office 365 Excel 文件
- ✅ Google Sheets 导出的 .xlsx 文件

### 不支持的格式
- ❌ Excel 97-2003 (.xls) - 需转换为 .xlsx 格式
- ❌ CSV 文件 - 需先导入 Excel 再导出为 .xlsx

## 最佳实践建议

1. **数据规范**: 保持释义格式统一，便于阅读和学习
2. **分册管理**: 根据学习阶段或难度划分多个词本文件
3. **定期备份**: 定期导出单词数据，防止数据丢失
4. **增量更新**: 新增单词时可创建增量文件单独导入

## 错误排除

### 常见问题

| 问题现象 | 可能原因 | 解决方案 |
|----------|----------|----------|
| 文件无法识别 | 文件格式错误 | 确保为 .xlsx 格式 |
| 部分单词未导入 | 必填字段为空 | 检查 A 列和 C 列数据 |
| 导入数量为零 | 标题行格式不正确 | 确保第一行为有效标题 |
| 导入速度慢 | 文件过大 | 拆分文件，控制在 1000 词以内 |
