# VocabularyApp - Android 英语单词学习应用技术文档

---

## 1. 项目概述

### 1.1 项目定位

**VocabularyApp** 是一款面向英语学习者的移动应用，旨在通过科学的学习方法和友好的用户界面，帮助用户高效记忆和掌握英语单词。应用采用原生 Android 开发技术，结合本地数据库实现数据持久化，支持离线学习场景。

### 1.2 核心价值

- **高效学习**: 通过卡片式交互和智能测试提升记忆效率
- **数据驱动**: 完整的学习数据分析和进度追踪
- **个性化体验**: 支持自定义词本导入和收藏管理
- **离线可用**: 本地数据存储，随时随地学习

### 1.3 技术栈

| 分类       | 技术            | 版本/说明        |
| ---------- | --------------- | ---------------- |
| 开发框架   | Android Native  | Java             |
| 数据库     | SQLite          | 内置             |
| UI 组件    | Material Design | AndroidX         |
| Excel 解析 | Apache POI      | 4.1.2            |
| 最低 SDK   | Android API     | 24 (Android 7.0) |
| 目标 SDK   | Android API     | 34 (Android 14)  |

### 2.2 模块详细说明

#### 2.2.1 单词学习模块

| 功能点   | 描述                               | 实现方式            |
| -------- | ---------------------------------- | ------------------- |
| 单词展示 | 大字体展示单词、音标、释义、例句   | CardView + TextView |

| 错题收藏 | 错误单词自动加入收藏       | 测试结束后批量处理       |

| 功能点   | 描述                  | 实现方式         |
| -------- | --------------------- | ---------------- |
| 单词列表 | 完整展示所有单词      | RecyclerView     |
| 实时搜索 | 支持模糊搜索          | SQLite LIKE 查询 |
| 快速收藏 | 列表中一键收藏        | 图标点击事件     |
| 批量导入 | 从 Excel 文件导入单词 | Apache POI 解析  |

#### 2.2.4 数据统计模块

| 功能点     | 描述                       | 实现方式               |
| ---------- | -------------------------- | ---------------------- |
| 学习统计   | 总单词数、已学单词数、进度 | 数据库聚合查询         |
| 测试记录   | 测试次数、平均得分、最高分 | 学习记录表统计         |
| 可视化展示 | 卡片式数据展示             | CardView + ProgressBar |

## 3. 数据库设计

### 3.1 数据库概述

- **数据库名称**: `VocabularyDB.db`
- **版本管理**: 通过 `SQLiteOpenHelper` 实现自动升级
- **存储位置**: 应用私有目录

### 3.2 数据表结构

#### 表 1: words（单词表）

| 字段名      | 类型    | 约束                      | 说明              |
| ----------- | ------- | ------------------------- | ----------------- |
| id          | INTEGER | PRIMARY KEY AUTOINCREMENT | 单词唯一标识      |
| word        | TEXT    | NOT NULL                  | 英语单词          |
| phonetic    | TEXT    |                           | 国际音标          |
| meaning     | TEXT    | NOT NULL                  | 中文释义          |
| example     | TEXT    |                           | 例句              |
| is_favorite | INTEGER | DEFAULT 0                 | 是否收藏（0/1）   |
| is_learned  | INTEGER | DEFAULT 0                 | 是否已学习（0/1） |
| study_count | INTEGER | DEFAULT 0                 | 学习次数          |
| difficulty  | INTEGER | DEFAULT 1                 | 难度等级（1-3）   |

#### 表 2: study_records（学习记录表）

| 字段名     | 类型    | 约束                      | 说明               |
| ---------- | ------- | ------------------------- | ------------------ |
| id         | INTEGER | PRIMARY KEY AUTOINCREMENT | 记录唯一标识       |
| timestamp  | INTEGER | NOT NULL                  | 学习时间戳         |
| word_count | INTEGER | DEFAULT 0                 | 学习单词数量       |
| test_score | INTEGER | DEFAULT 0                 | 测试得分           |
| test_total | INTEGER | DEFAULT 0                 | 测试题目总数       |
| type       | TEXT    | NOT NULL                  | 类型（study/test） |

#### 表 3: statistics（统计数据表）

| 字段名      | 类型    | 约束                      | 说明       |
| ----------- | ------- | ------------------------- | ---------- |
| id          | INTEGER | PRIMARY KEY AUTOINCREMENT | 统计项标识 |
| key         | TEXT    | UNIQUE NOT NULL           | 统计键名   |
| value       | TEXT    |                           | 统计值     |
| update_time | INTEGER |                           | 更新时间戳 |

### 3.3 核心 SQL 语句

```sql
-- 创建单词表
CREATE TABLE IF NOT EXISTS words (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    word TEXT NOT NULL,
    phonetic TEXT,
    meaning TEXT NOT NULL,
    example TEXT,
    is_favorite INTEGER DEFAULT 0,
    is_learned INTEGER DEFAULT 0,
    study_count INTEGER DEFAULT 0,
    difficulty INTEGER DEFAULT 1
);

-- 创建学习记录表
CREATE TABLE IF NOT EXISTS study_records (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    timestamp INTEGER NOT NULL,
    word_count INTEGER DEFAULT 0,
    test_score INTEGER DEFAULT 0,
    test_total INTEGER DEFAULT 0,
    type TEXT NOT NULL
);

-- 创建索引
CREATE INDEX IF NOT EXISTS idx_words_word ON words(word);
CREATE INDEX IF NOT EXISTS idx_words_favorite ON words(is_favorite);
CREATE INDEX IF NOT EXISTS idx_records_type ON study_records(type);
```

## 4. 界面设计

### 4.1 设计原则

1. **视觉层次**: 通过卡片阴影和间距建立清晰的视觉层级
2. **色彩和谐**: 采用莫兰迪色系，柔和舒适适合长时间学习
3. **交互反馈**: 提供清晰的点击反馈和状态变化
4. **响应式布局**: 适配不同屏幕尺寸

### 4.2 界面原型

## 6. API 接口说明

### 6.1 数据库操作接口

#### Word 相关操作

| 方法名                                 | 功能描述       | 参数                           | 返回值           |
| -------------------------------------- | -------------- | ------------------------------ | ---------------- |
| `addWord(Word word)`                   | 添加单词       | `Word`: 单词对象               | `long`: 插入的ID |
| `getAllWords()`                        | 获取所有单词   | 无                             | `List<Word>`     |
| `searchWords(String keyword)`          | 搜索单词       | `keyword`: 搜索关键词          | `List<Word>`     |
| `getFavoriteWords()`                   | 获取收藏单词   | 无                             | `List<Word>`     |
| `updateFavorite(int id, boolean flag)` | 更新收藏状态   | `id`: 单词ID, `flag`: 是否收藏 | `int`: 影响行数  |
| `updateDifficulty(int id, int level)`  | 更新难度等级   | `id`: 单词ID, `level`: 难度    | `int`: 影响行数  |
| `getTotalWordCount()`                  | 获取单词总数   | 无                             | `int`: 数量      |
| `getLearnedWordCount()`                | 获取已学单词数 | 无                             | `int`: 数量      |

#### StudyRecord 相关操作

| 方法名                               | 功能描述     | 参数                    | 返回值           |
| ------------------------------------ | ------------ | ----------------------- | ---------------- |
| `addStudyRecord(StudyRecord record)` | 添加学习记录 | `StudyRecord`: 记录对象 | `long`: 插入的ID |
| `getTestCount()`                     | 获取测试次数 | 无                      | `int`: 次数      |
| `getAverageScore()`                  | 获取平均得分 | 无                      | `double`: 平均分 |
| `getHighestScore()`                  | 获取最高分   | 无                      | `int`: 最高分    |

### 6.2 Excel 导入接口

| 方法名                        | 功能描述        | 参数           | 返回值              |
| ----------------------------- | --------------- | -------------- | ------------------- |
| `importFromExcel(Uri uri)`    | 从Excel导入单词 | `uri`: 文件URI | `int`: 导入数量     |
| `isValidExcelFormat(Uri uri)` | 验证Excel格式   | `uri`: 文件URI | `boolean`: 是否有效 |

## 8. 数据安全

### 8.1 数据保护

- 数据库文件存储在应用私有目录，其他应用无法访问
- 敏感数据（如学习记录）仅本地存储，不上传云端
- 支持数据导出备份功能

### 8.2 错误处理

- 数据库操作异常捕获和日志记录
- Excel导入失败时的错误提示
- 空数据状态的友好提示
